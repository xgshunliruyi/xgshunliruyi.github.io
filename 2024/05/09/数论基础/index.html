辗转相除

带余除法

设$a, b$是两个整数，其中$b \neq 0$，则存在两个唯一的整数$q, r$，使得
$$a = bq+r, \quad 0 \leq r<|b|$$
称式中$q$为$a$被$b$除得出的不完全商，$r$为$a$被$b$除得出的余数，也称为非负最小余数，通常记作$<a>_{b}=r$

辗转相除法

设整数$a,b(\neq 0)$，由带余除法得下列等式
$$a=bq_{1}+r_{1}, \quad 0<r_{1}<|b|$$
$$b=r_{1} q_{2}+r_{2}, \quad 0<r_{2}<r_{1}$$
$$…$$
$$r_{n-2}=r_{n-1} q_{n}+r_{n}, \quad 0<r_{n}<r_{n-1}$$
$$r_{n-1}=r_{n} q_{n+1}+r_{n+1}, \quad r_{n+1}=0$$
因为$|b| > r_{1} > r_{2} > …$，故经过有限次带余除法后，总可以得到一个余数是零。上述过程就称为辗转相除

最大公因数

定义

设$a_{1},a_{2},…,a_{n}$是n个不全为0的整数，如果整数$d$是它们之中每一个的因数，那么$d$就称为$a_{1},a_{2},…,a_{n}$的一个公因数。整数$a_{1},a_{2},…,a_{n}$的最大公因数记作$(a_{1},a_{2},…,a_{n})$。如果$(a_{1},a_{2},…,a_{n})=1$，就称$a_{1},a_{2},…,a_{n}$互素或互质
注意：$(a_{1},a_{2},…,a_{n})=1$，但其中任意两个或任意小于n个整数不一定互素

定理1

设$a,b,c$是任意三个不全为0的整数，且$a=bq+c$，其中$q$是整数，则$(a,b)=(b,c)$

辗转相除求最大公因数

根据上述定理，对任意整数$a>0,b>0$，作辗转相除，则最后一个非零余数$r_{n}=(a,b)$
证明
$$a=bq_{1}+r_{1}, \quad 0<r_{1}<|b|$$
$$b=r_{1} q_{2}+r_{2}, \quad 0<r_{2}<r_{1}$$
$$…$$
$$r_{n-2}=r_{n-1} q_{n}+r_{n}, \quad 0<r_{n}<r_{n-1}$$
$$r_{n-1}=r_{n} q_{n+1}+r_{n+1}, \quad r_{n+1}=0$$
则由上述定理，$(a,b)=(b,r_{1})=(r_{1},r_{2})=…=(r_{n-1},r_{n})=r_{n}$

定理2

对任意不全为0整数$a,b$，存在整数$u,v$，使得$(a,b)=au+bv$
证明
对整数$a,b$作上述辗转相除后回代
$$r_{n}=r_{n-2}-r_{n-1}q_{n}$$
$$=r_{n-2}-(r_{n-3}-r_{n-2}q_{n-1})q_{n}$$
$$=…=au+bv$$

最小公倍数

定义

设$a_{1},a_{2},…,a_{n}$是n个整数(n>1), 若整数m是这n个数每一个数的倍数，则m就称为这n个数的公倍数。$a_{1},a_{2},…,a_{n}$的最小公倍数记作$[a_{1},a_{2},…,a_{n}]$

计算方法

$$[a,b]=\frac{|ab|}{(a,b)}$$

二元一次不定方程

二元一次不定方程形如
$$ax+by=c$$
其中$a,b,c$都是给定的整数，且$ab \neq 0$
定理
上述二元一次不定方程有整数解的充要条件是$(a,b)|c$, 且其全部整数解可以表示为
$$x=x_{0}+\frac{b}{(a,b)}t, \quad y=y_{0}-\frac{a}{(a,b)}t, \quad t=0,±1,…$$
其中$x_{0},y_{0}$是其任意一组特解

同余

定义

给定一个正整数$m$，如果用$m$去除两个整数$a, b$所得的余数相同，则称$a, b$对模数$m$同余，并称$a \equiv b \ (mod \ m)$为同余式。

性质

  • $a \equiv b \ (mod \ m)$的充要条件是$m|a-b$
  • 如果$a \equiv b \ (mod \ m), \quad \alpha \equiv \beta \ (mod \ m)$, 则有
    • $ax+ \alpha y \equiv bx+ \beta y \ (mod \ m)$, 其中$x,y$为任意整数
    • $a \alpha \equiv b \beta \ (mod \ m)$
    • $a^{n} \equiv b^{n} \ (mod \ m)$
    • $f(a) \equiv f(b) \ (mod \ m)$, 其中$f(x)$是任意整系数多项式
  • 如果$ac \equiv bc \ (mod \ m)$, 且$(m,c)=d$, 则$a \equiv b \ (mod \ \frac{m}{d})$
  • 如果$a \equiv b \ (mod \ m_{i}), \quad i=1,2,…,n$, 则$a \equiv b \ (mod \ [m_{1},m_{2},…,m_{n}])$

一次同余方程

一次同余方程是指
$$ax \equiv b \ (mod \ m)$$
其中$a !\equiv 0 \ (mod \ m), m>1$
如果$x_{0}$满足上式, 则$x \equiv x_{0} \ (mod \ m)$称为同余方程的解
注意：不同的解是指对模数$m$互不同余的解

定理

设$a \neq 0, m>0$, 则同余方程$ax \equiv b \ (mod \ m)$有解的充要条件是$(a,m)|b$, 在有解的条件下，它有$(a,m)$个解，且它们是$x \equiv x_{0}+\frac{m}{(a,m)}t \ (mod \ m), \quad t=0,1,…,(a,m)-1$。其中$x_{0}$是同余方程的任意一个解

逆

特别地，对于同余方程$ax \equiv 1 \ (mod \ m)$, 当且仅当$(a,m)=1$时有解且恰有一个解，称这个解为$a$模$m$的逆，记为$a^{-1}$

解一次同余方程的一般方法

由$2.4$的定理可以求出唯一的整数$u,v$满足$(a,m)=au+mv$，又$(a,m) \le m$，则$(a,m) \equiv au \ (mod \ m)$，又$(a,m)|b$，所以$x \equiv u \frac{b}{(a,m)} \ (mod \ m)$是同余方程的一个解，在根据定理$5.4$即可求出通解

整数分解定理

  • 若$p$是素数，$a$是任意整数，则有$p|a$或$(p,a)=1$
  • 若$p$是素数，$p|ab$，则$p|a$或$p|b$

证明
假设$p|ab$，但$p \nmid a$，因为$p$是素数，从而$(p, a)=1$
根据定理，一定存在整数$x,y$满足$ax+py=(p,a)=1$
等式两边乘$b$得$abx+pby=b$
又由于$p|ab$，从而$ab=kp, \quad k \in N^{+}$
故$kpx+pby=b$，即$p(kx+by)=b$，所以$p|b$
证毕
整数分解定理（算数基本定理）
任何一个大于1的整数都能唯一的分解成素数的乘积，即
$$\forall a > 1, \ a=p_{1}p_{2}…p_{n}, \quad p_{1} \le p_{2} \le … \le p_{n}$$
其中$p_{1}, \ p_{2} \ , \ … \ , \ p_{n}$都是素数
如果合并相同的素数，则有
$$a = p_{1}^{l_{1}}p_{2}^{l_{2}}…p_{r}^{l_{r}}, \quad p_{1}<p_{2}<…<p_{n}, \quad l_{i} \ge 1$$

欧拉函数

定义

欧拉函数$\phi (n)$是一个定义在正整数集合上的函数，$\phi (n)$的值等于序列$0,1,…,n-1$中与$n$互素的数的个数
规定：$0$与任何数都互素，$1$与任何数都不互素。即$\phi (1)=1, \phi (2) = 1$

性质

  • 若$p$是素数，则$\phi (p) = p-1$
  • $(m_{1}, m_{2})=1$时，$\phi (m_{1}m_{2}) = \phi (m_{1}) \phi (m_{2})$
  • $p$为素数，$l \in N^{+}$时，$\phi (p^{l})=p^{l}-p^{l-1} = p^{l-1}(p-1)$

计算

根据欧拉函数的基本性质以及算术基本定理，可得欧拉函数的计算公式
$m = p_{1}^{l_{1}}p_{2}^{l_{2}}…p_{k}^{l_{k}}$，$p_{i}$为素数，$l_{i}>0$
$$
\begin{array}{l}
\phi (m) \\
= \phi (p_{1}^{l_{1}}p_{2}^{l_{2}}…p_{k}^{l_{k}}) \\
= \phi (p_{1}^{l_{1}}) \phi(p_{2}^{l_{2}})… \phi(p_{k}^{l_{k}}) \\
= \prod_{i=1}^{k} p_{i}^{l_i-1}(p_i-1) \\
= \prod_{i=1}^{k} p_i^{l_i}(1-\frac{1}{p_i}) \\
= \prod_{i=1}^{k} p_i^{l_i} \prod_{i=1}^{k} (1-\frac{1}{p_i}) \\
= m\prod_{i=1}^{k} (1-\frac{1}{p_i})
\end{array}
$$

欧拉定理

若$(a,m)=1$，则$a^{\phi (m)} \equiv 1 \ (mod \ m)$
由定理推得：若$(a,m)=1$，则$a^{\phi (m) + 1} \equiv a \ (mod \ m)$

费马小定理

在欧拉定理中取$m=p$为素数，则$a^p \equiv a \ (mod \ p)$

孙子定理

一次同余方程组的形式如下：
$$
\begin{array}{l}
x \equiv b_1 \ (mod \ m_1) \\
x \equiv b_2 \ (mod \ m_2) \\
… \\
x \equiv b_k \ (mod \ m_k)
\end{array}
$$
其中$m_1, m_2, … ,m_k$是$k$个两两互素的正整数
$m = m_1 m_2 … m_n = m_i M_i \ (i=1,2,…,k)$，则上述一次同余方程组具有唯一解
$$x \equiv M_1^{'} M_1 b_1 + M_2^{'} M_2 b_2 + … + M_k^{'} M_k b_k \ (mod \ m)$$
其中$M_i^{'} M_i \equiv 1 \ (mod \ m_i) \ (i=1,2,…,k)$
