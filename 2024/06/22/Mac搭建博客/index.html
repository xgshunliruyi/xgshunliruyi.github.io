在macOS上搭建hexo个人博客

Sliver Eagle
2024-06-22
搭建个人博客
macOS, 搭建个人博客

研究生上岸，最近换了一台MacBook，不得不说macOS的颜值是真的很高，从windows换到macOS第一感觉就是GUI太漂亮了，最近打算重新搭建自己的博客，记录一下过程。
假设我们拿到的是一台全新的MacBook。

配置环境

第一部分是配置自己的电脑环境，需要配置和安装的软件有

  1. homebrew
  2. git
  3. node
  4. hexo

安装Homebrew

这是macos的软件包管理工具，使用它下载和管理软件、依赖等都非常方便，如果可以的话，尽量都使用homebrew安装软件，这是因为在我们卸载软件的时候，homebrew会跟踪软件并且自动的帮我们清理掉残留的文件和依赖

接下来我们使用下面的命令验证是否安装成功

这时可能显示zsh:command not found:brew，但执行/opt/homebrew/bin/brew --version可以显示版本号，原因是homebrew的安装路径并没有添加到系统的PATH环境变量中。
解决办法：打开文件nano ~/.zshrc，在文件的末尾添加export PATH="/opt/homebrew/bin:$PATH"，最后重新加载配置文件source ~/.zshrc。
再次验证就好啦。

安装Git

你可以先使用git --version来查看Git是否已经安装，如果没有，使用brew install git安装即可。

安装Node

同样，你可以先用node -v和npm -v来查看是否已经安装，如果没有，请使用brew install node

配置主题

现在我们一个裸奔的博客就搭建完成了，是可以写文章发文章的，但是接下来的美化操作我就略过了，因为配置文件_config.yml的注释已经非常明了了，如果不清楚，可以看一下hexo的文档
另外，主题的选择和美化也略过了，毕竟萝卜青菜各有所爱，大家可以选择自己喜欢的主题就好，我使用的主题是cactus，简约风格，而且配置非常简单，不需要安装任何额外的插件，配置文件的注释也非常详细，最重要的是它对于公式的支持很好，只需要在配置文件中将公式支持改为true即可，我以往使用的一些主题对公式的支持都不是很好，不仅需要更换渲染器，还需要修改一些文件，很不方便。
